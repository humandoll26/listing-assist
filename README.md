# Listing Assist

商品マスター・出品データ・在庫管理を扱う、ブラウザ内完結型の出品一覧ツールです。

## スマホ実機確認

GitHub Pages: <https://humandoll26.github.io/listing-assist/>

- データは端末ごとのIndexedDBに保存されます。
- PCとスマートフォンのローカルデータは自動では共有されません。
- Google Drive同期を試すときは、最初にテスト用バックアップを使用してください。
- QR棚登録のカメラ確認にはHTTPSのPages URLを使用してください。

## 公開対象

Pagesでは `index.html`、`styles.css`、`app.js`、`bootstrap.js`、`js/data/` を配信します。
レビュー資料、実行ログ、生成物、デバッグ用CSVはリポジトリに含めません。
